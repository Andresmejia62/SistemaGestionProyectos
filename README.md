## 1. Descripción General del Sistema
El **Sistema de Gestión de Proyectos** es una solución de base de datos relacional desarrollada en Microsoft SQL Server. Está diseñado para centralizar y administrar el ciclo de vida completo de iniciativas tecnológicas y de infraestructura. 

El sistema permite un control estricto de:
* **Recursos Humanos:** Colaboradores, asignaciones específicas y control de sobrecarga de trabajo.
* **Alcance del Proyecto:** Gestión de hitos cronológicos, tareas operativas y entregables validados.
* **Salud Financiera:** Control de presupuestos aprobados, órdenes de adquisición y desglose de gastos reales facturados.
* **Gestión de Riesgos:** Registro de incidencias críticas junto con sus respectivos planes de mitigación.
* **Seguridad y Auditoría:** Gestión de usuarios del sistema e historial automático de transacciones (`INSERT`, `UPDATE`, `DELETE`).
